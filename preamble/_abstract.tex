\textoResumo {

Este trabalho de conclusão de curso tem como objetivo o estudo de estações de carregamento de veículos elétricos (EVSE), com foco em seu \textit{software}. A implementação utiliza o protocolo OCPP, ao qual caminha para se tornar padrão oficial, assim como MODBUS para integrar os diversos dispositivos utilizados no projeto.
}

\palavrasChave {Engenharia Elétrica, Veículos Elétricos, Eletroposto, EVSE, Comunicação, Sistemas Embarcados}

\textAbstract {

This final project aims the study of EVSEs (Electric Vehicle Supply Equipment), focusing on it's software. It implements the OCPP protocol, which soon will become standard for EVSEs, as it implements MODBUS to integrate all devices used in the project.

}

\keywords {electronics, evse, electric vehicles, charging stations, embedded systems, beaglebone}
