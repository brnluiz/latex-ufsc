\textoResumo {

Este trabalho de conclusão de curso tem como objetivo o estudo de estações de carregamento de veículos elétricos (EVSE), com foco em sua implementação de \textit{software}. A implementação utiliza o protocolo OCPP que, ao que tudo indica, irá se tornar o padrão oficial para EVSEs, assim como MODBUS para integrar os diversos dispositivos utilizados no projeto. A estação em estudo segue o padrão Europeu e possibilita carregamentos modo 2 e modo 3 (\textit{Mennekes}).
}

\palavrasChave {Engenharia Elétrica, Veículos Elétricos, Eletroposto, EVSE, Comunicação, Sistemas Embarcados}

\textAbstract {

This Final Year Project aims the study of EVSEs (Electric Vehicle Supply Equipment), focusing on its software implementation. It implements the OCPP protocol, which may become standard for EVSEs, as it implements MODBUS to integrate all devices used in the project. The station follows the European standard and enables mode 2 and 3 charging.

}
